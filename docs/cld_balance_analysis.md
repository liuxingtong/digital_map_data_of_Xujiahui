# CLD回路平衡性诊断与精简方案

---

## 一、当前问题诊断

### 1.1 回路极性分布

CLD中，一条回路的类型由**回路内所有边极性之积**决定：
- 负边数为**奇数** → 平衡回路（B）
- 负边数为**偶数**（含0）→ 强化回路（R）

当前六条"回路"的实际极性：

| 回路 | 闭合路径 | 负边数 | 实际类型 | 问题 |
|------|---------|-------|---------|------|
| R1 | N01(+)→N02(+)→N03(+)→N01 | 0 | 强化 R | ✓ 正确 |
| R2 | N04→N05→N02→N03→N01（注入R1） | 0 | 强化 R | ⚠ 未自我闭合，是链条非回路 |
| R3 | N07→N09(+)→N02→N03→N01（注入R1） | 0 | 强化 R | ⚠ 同上，未闭合 |
| B1 | N10→N11(-)→N13→N02→N10(?) | — | **无闭合** | ❌ N02没有边指向N10，B1根本未闭合 |
| R4 | N14(+)→N15(+)→N16(+)→N14 | 0 | 强化 R | ✓ 闭合，但无内置限制 |
| R5 | N18(+)→N02(+)→N19(+)→N20(+)→N18 | 0 | 强化 R | ✓ 闭合，但无内置限制 |

**核心问题：**
- R2和R3是向R1注入的**链条**，不是独立闭合回路
- B1实际上**没有闭合**——系统中找不到从N02回到N10的边
- 全部闭合回路（R1、R4、R5）均为强化回路，**零平衡回路**
- 当前29条边中，负向边仅4条（E12、E13、E16、E29），占比14%

---

### 1.2 被错误标注为负向的关系

**B1（交通强度 ↔ 步行意愿）：**

现有描述将B1称为"平衡回路"，原因是"步行意愿低→聚集减少→间接降低对交通的需求"。但：
1. N13→N02的边是(+)：步行意愿→聚集是正向
2. N02不连向N10，回路未闭合
3. 即使补上N13→N10(−)使其闭合：N10→N11(+)→N13(−)→N10(−)，负边数=2（偶数），**仍是强化回路R**——城市规划中著名的"交通诱导需求"恶性循环

> **诊断结论：B1实际上是一个指向坏均衡的恶性强化回路R_bad（高车流→低步行→更多开车→更高车流），而非平衡回路。当前文档的命名是错的。**

| 节点对 | 结构关系 | 问题 | 建议 |

## 三、新增平衡回路（3条）

### B2 — 拥挤自调节（新增）

**新增边：** N02 → N08/原N09 街道舒适度 **(−)**

**机制：** 老年人聚集强度过高时，空间拥挤感上升，噪声增加，私密性降低，反而降低舒适度——这是公共空间研究中有充分实证的"去中心效应"（crowding effect）。

**闭合回路：**
```
N09舒适度 ──(+)──→ N02聚集强度
N02聚集强度 ──(−)──→ N09舒适度  ← 新增
```
负边数：1（奇数）→ **平衡回路 B2** ✓

**行为意义：** 系统不会无限聚集，高密度会自然触发舒适度下降，减缓进一步聚集。这是空间有机调节的内生机制，不需要外部干预。

---

### B3 — 认知超载自限（新增）

**新增边：** N14/原N14 空间认知复杂度指数 → N17/原N17 环境生理压力指数 **(+)**

**机制：** 空间认知复杂度存在倒U型效应——适度复杂提供训练，过度复杂产生认知超载（cognitive overload），认知超载是一种生理压力，同属"环境对人体的胁迫"，应纳入N17指数。这有充分的认知神经科学文献支持（与Avila-Palencia 2025论文的CLD中"cognitive reserve"—"physiological stress"链路一致）。

**闭合回路：**
```
N14复杂度 ──(+)──→ N15认知储备激活潜力     ← 已有E15
N15        ──(+)──→ N16主动探索意愿         ← 已有E17
N16        ──(+)──→ N14复杂度               ← 已有E18（R4回路）

新增路径：
N14复杂度 ──(+)──→ N17环境生理压力  ← 新增
N17        ──(−)──→ N15认知储备激活  ← 已有E16
→ 回路：N14→(+)N17→(−)N15→(+)N16→(+)N14
```
负边数：1（奇数）→ **平衡回路 B3** ✓

**行为意义：** R4回路不会无限自我强化——当N14极高时，认知超载压力（通过N17）会反向抑制N15，探索意愿N16随之下降，对N14高复杂区域的使用减少。R4和B3共同构成"倒U型"的系统行为，而不是单纯的线性放大。

**附加意义：** N17现在不只接收交通/噪声/污染输入，还接收认知超载输入。这使N17成为更完整的"环境对老年人总体胁迫指数"，与Avila-Palencia论文的框架更精确对应。

---

### B4 — 干预疲劳（新增）

**新增边：** N18/原N18 人机共生干预指数 → N19/原N19 行为数据密度 **(−)**

**机制：** 当人机干预密度过高或持续时间过长，产生"设备疲劳"（technology habituation）——使用者对装置的新奇感下降，交互频率降低，数据密度反而减少。这是技术类干预的普遍规律（参照用户行为研究中的"habituation效应"）。

**闭合回路：**
```
N19行为数据密度 ──(+)──→ N20干预优化  ← 已有E21
N20            ──(+)──→ N18人机干预指数 ← 已有E22
N18            ──(−)──→ N19行为数据密度 ← 新增
```
负边数：1（奇数）→ **平衡回路 B4** ✓

**行为意义：** R5数据闭环不会无限正向累积。干预效能有边际递减，迫使系统不断引入新奇性（换装置、换内容、换时段）才能维持数据密度。这正是§七中"每周重排""四日构型""三时段显示"等可变机制的系统动力学依据——可变性是B4平衡压力的直接设计响应。

---

### R_bad — 交通诱导（原B1重命名）

**原B1的诊断结论：** 添加闭合边 N13步行意愿 → N09车流量 **(−)** 后：

```
N09车流量 ──(+)──→ N11噪声与污染
N11       ──(−)──→ N13步行意愿
N13       ──(−)──→ N09车流量   ← 新增闭合边
```
负边数：2（偶数）→ **强化回路** ——指向坏均衡的恶性循环

**重新命名为 R_bad（交通诱导恶性循环）：**
高车流→高噪声污染→步行意愿下降→人们更依赖机动车→车流进一步上升。这是城市规划中著名的"诱导交通需求"（induced demand）效应，全球各地有充分实证。

> **这个修正非常重要：** 原文档将其命名为"平衡回路B1"，暗示系统有自我调节机制；真实情况是这是一个会自我强化的坏结构，**需要外生干预（N18人机共生指数对N09的负向作用）才能打破**，而不会自然趋于平衡。

**对应的真正平衡机制：**
已有边 E29：N18 → N17(−) 可以扩展为 N18 → N09(−)
机制：原型C地面光谱路径的路径引导系统，将行人流量引导离开高车流路段，间接降低该路段的交通压力；若系统接入交通管理后端，甚至可直接触发某些路段的限速/限流措施。

## 六、系统行为解读（修订后）

### 系统现在有了内置的"刹车机制"

**B2拥挤调节：** R1（社交活力）的强化不会无限累积——当N02聚集强度升高，R3中的N09舒适度会通过E26被反压，形成对R1的自然限制。最终R1+B2将在某个"适宜聚集密度"上趋于动态平衡，而不是单向爆炸或崩溃。

**B3认知超载：** R4（认知恢复）的正反馈现在有了内置天花板——复杂度的过度提升会通过N17反压认知储备，让R4在"认知训练最优区间"（而非最大值）附近稳定。这正好对应了文档中反复提到但一直没有在CLD中编码的"倒U型效应"。

**B4干预疲劳：** R5（人机共生）的数据闭环不会永续增长——habituation效应施加的压力迫使系统持续引入新奇性（对应六原型的可变机制设计），这才是这些原型"必须可变"的系统动力学依据，而不只是设计美学偏好。

**R_bad的诊断意义：** 原来以为系统有一个"平衡"交通的机制，现在明确：没有。交通是系统中唯一需要**外生强制干预**（N16→N09，E30）才能打破的恶性循环。这对设计决策有直接含义：原型C（地面光谱路径）的路径引导功能，不只是"改善步行体验"，而是**系统结构上破解R_bad的唯一设计杠杆**。
