## 二、街景数据

### 2.1 原始字段

街景图像语义分割 CSV 含各语义类别的像素占比，如：

- `vegetation`, `terrain`, `sky`, `water`
- `building`, `wall`, `fence`, `barrier`
- `sidewalk`, `crosswalk-plain`, `pedestrian area`, `road`
- `car`, `bus`, `truck`, `motorcycle`
- `street light`, `traffic light`, `bench`, `traffic sign`
- 等 7 大语义类别（nature, built, road, sign, furniture, people, vehicle）

### 2.2 衍生指标

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| **GVI** | 绿视率 | vegetation + terrain |
| **SVF** | 天空可见度 | sky |
| **Enclosure** | 围合度 | building + wall + fence + barrier |
| **BVI** | 蓝视率 | water |
| **Ped_Infra** | 步行基础设施 | sidewalk + crosswalk + curb cut + pedestrian area + lane marking |
| **Motor_Pressure** | 机动车压力 | car + bus + truck + motorcycle + road |
| **Ped_Safety** | 步行安全感 | (Ped_Infra + lights - Motor_Pressure) 归一化 |
| **Shade_Comfort** | 遮蔽舒适度 | (vegetation + building + bridge + tunnel - sky) 归一化 |
| **Shannon_H** | Shannon 多样性 | -Σ(pᵢ·ln(pᵢ))，按 7 大语义类别 |
| **Simpson_D** | Simpson 多样性 | 1 - Σ(pᵢ²) |
| **Complexity** | 视觉复杂度 | 非零类别数 × Shannon_H，归一化 |
| **Fascination** | ART 魅力 | vegetation + water + terrain + 0.5×sky |
| **Being_Away** | ART 远离 | 1 - (car + bus + truck + road + billboard + CCTV) |
| **Coherence** | ART 延展 | 基于 Shannon 的倒 U 型，中等多样性最优 |
| **Ped_Friendly** | ART 兼容 | sidewalk + bench + pedestrian area + street light - barrier - fence |
| **ART_Score** | ART 综合恢复潜力 | 0.3×Fascination + 0.25×Being_Away + 0.2×Coherence + 0.25×Ped_Friendly |

## 四、POI 数据

### 4.1 原始字段

| 字段 | 含义 |
|------|------|
| id, name, type, address | 标识与分类 |
| lng, lat (或 wgs84Lng/wgs84Lat) | 坐标 |
| group | 类别（咖啡店、文化地标、医疗保健等） |
| rating | 评分（部分有） |

### 4.2 衍生指标

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| `rating_numeric` | 数值型评分 | rating 转 float，缺失填 0 |
| `density` | 点密度 | 恒为 1，供 KDE 加权 |
| `category` | 细分类别 | 从 type 提取最后一段 |
| `poi_count` | 半径内 POI 数 | 边中点 100m 半径内计数（路网挂载） |
| 按类别聚合 | 各 category 数量 | `aggregate_poi_by_category_near` |

### 4.3 数据文件

咖啡店、全部 POI、交通设施、文化地标、医疗保健、体育休闲、风景名胜、购物菜场、生活服务、社交餐饮、金融住宿政府、商务产业、银发极客/科技体验、独处品质空间等。

## 六、用地数据

### 6.1 原始字段

OSM 多边形，含 `landuse`、`leisure`、`natural` 等标签，提取为 `landuse_type`。

### 6.2 衍生指标（全域）

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| `shannon_entropy` | Shannon 多样性熵 | 面积加权 H = -Σ(pᵢ·ln(pᵢ)) |
| `green_space_rate_pct` | 绿地率 | 绿地面积 / 总面积 × 100% |
| `mixed_use_ratio_pct` | 混合用途比例 | 250m 网格中 ≥2 类用地的网格占比 |

### 6.3 衍生指标（网格级，250m）

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| `shannon` | 网格 Shannon 熵 | 网格内类型计数的熵 |
| `green_rate` | 网格绿地率 | 网格内绿地类型占比 |
| `n_types` | 网格用地类型数 | 网格内不同 landuse_type 数量 |

## 七、路网数据

### 7.1 原始字段（OSM 边属性）

| 字段 | 含义 |
|------|------|
| `highway_type` | 道路类型（primary, secondary, footway, path 等） |
| `name` | 路名 |
| `length` | 路段长度（米） |
| `maxspeed` | 限速（km/h） |
| `lanes` | 车道数 |
| `oneway` | 是否单向 |

### 7.2 衍生指标（图论）

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| `alpha` | α 指数 | 实际环数 / 最大可能环数 |
| `beta` | β 指数 | 边数 / 节点数 |
| `gamma` | γ 指数 | 实际边数 / 最大可能边数 |
| `avg_edge_length_m` | 平均路段长度 | 米 |
| `intersection_count` | 交叉口数 | 度 ≥ 3 的节点数 |
| 交叉口密度 | 个/km² | intersection_count / 面积 |

### 7.3 衍生指标（汇总）

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| `pedestrian_ratio_pct` | 步行专用路比例 | footway/path/steps/pedestrian 边占比 |
| `motor_ratio_pct` | 机动车干道比例 | primary/secondary/trunk 边占比 |
| `lanes_coverage_pct` | lanes 数据覆盖率 | 有 lanes 属性的边占比 |

### 7.4 路网挂载（边属性）

通过 `analysis/road/weights.py` 可挂载到边中点：

| 挂载函数 | 挂载属性 | 数据源 |
|----------|----------|--------|
| `attach_streetview_scores` / `attach_streetview_scores_multi` | streetview_{indicator} | 街景最近点插值 |
| `attach_poi_density` / `attach_poi_by_category` | poi_count, poi_social, poi_medical, poi_culture | 边中点 100m 半径内 POI 数 |
| `attach_population` / `attach_population_multiage` | population, pop_65plus, pop_15_59, pop_0_14 | 人口栅格采样 |
| `attach_landuse` | landuse_shannon, landuse_green_rate, landuse_n_types | 用地网格（250m）最近邻 |
| `compute_edge_traffic_pressure` | edge_traffic_pressure | 边级车流量/噪声代理 |
